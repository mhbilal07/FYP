# FYP

A frontend project built with Vite, React, TypeScript, and Tailwind CSS.

## Features

- Modern React + TypeScript setup
- Vite-based development workflow
- Tailwind CSS styling
- Reusable UI component structure

## Project Structure

```text
frontend/
  src/
    app/
      App.tsx
      components/
    main.tsx
    styles/
  package.json
  vite.config.ts
```

## Getting Started

1. Install dependencies:
   ```bash
   cd frontend
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open the local URL shown in the terminal.

## Scripts

- `npm run dev` — start the development server
- `npm run build` — build the project for production
- `npm run preview` — preview the production build

## Notes

- The project uses a shadcn-style UI component structure under `frontend/src/app/components/ui`.
- Generated build artifacts and dependencies are ignored by Git.
